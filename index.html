What is Steak?

  • Acceptance testing framework
  • Ruby / Rspec / webrat
  • Inspired by Cucumber, Story Runner? Maybe

Why would I use it?

  • Fast and easy to write
  • Fast execution
  • Sometimes I feel Cucumber is hard and not worth it

Where do I find good Steak?

  • Capital Gill, Grill 23, Mortons???
  • It's much cheaper on github
  • http://github.com/cavalle/steak

How do I install Steak?

It's a gem, silly

          gem install steak

Then (in your test helper):

          require 'steak'

What does Steak look like?

It's Cucumber-rspec-esque:

  feature "Login" do
    background do
      create_user :login => "jpowers"
      login_as "jpowers"
    end
    scenario "should see my login" do
      visit '/'
      page.should have_css(".login", :text => "jpowers")
    end
  end

Rails Integration

Easy

          script/generate steak (--webrat)
        
          script/generate acceptance_spec cool_new_feature
        
          rake spec:acceptance
